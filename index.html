Projet Sécurité Informatique

Analyse et Exploitation des Vulnérabilités Web avec DVWA, Burp Suite et Kali Linux

Voir détails
Objectifs du Projet :

Ce projet m'a permis d'approfondir mes connaissances en data analysis pour la gestion des vulnérabilités, de renforcer mes compétences en cybersécurité, et de mieux comprendre les pratiques de sécurité à implémenter pour protéger les applications contre les attaques.

Tâches effectuées :
  • Injection SQL : J'ai pu exploiter une faille SQL dans DVWA, montrant comment des données non sécurisées peuvent être utilisées pour compromettre une base de données.
  • Cross-Site Scripting (XSS) : Grâce à Burp Suite, j'ai intercepté et modifié des requêtes pour injecter des scripts malveillants dans les pages.
  • CSRF (Cross-Site Request Forgery) : Simulation d'une attaque de type CSRF, exploitant la confiance de la session utilisateur pour exécuter des actions non autorisées.
  • Analyse des données récupérées lors des injections et des attaques simulées avec MySql
